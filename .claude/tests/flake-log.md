# Flake Log

## Format

```

**Package**: <e.g., packages/core>
**Symptom**: <what fails non-deterministically>
**Hypothesis**: <what might be wrong>
**Owner**: <agent or name>
**Deadline**: YYYY-MM-DD (1 week from entry)
**Resolution**: fixed | removed | escalated
```
